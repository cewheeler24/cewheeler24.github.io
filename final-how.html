Clearly Labeled Sections

Each portion of the content of your website should be labeled using h tags. For example, this section is labeled with the h2 tag, "Clearly Labeled Sections". This way, screen reader users can navigate the website easier by skipping through to the content they actually care about.

Keyboard/Screen Reader Usability

All interactive elements, such as drop down menus, should be accessible with keyboard-only use. Additionally, the use of the interaction should be clearly read with a screen reader. Additionally, all images should have consise alt text

Color Contrast

Text and background colors should have sufficient contrast to ensure readability for users with visual impairments. Tools like the WebAIM Contrast Checker can help determine if your color choices meet accessibility standards.
